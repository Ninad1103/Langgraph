Usage
Ollama llms are required to run the agents. Install it from https://ollama.com/.
Basic agents, graphs and workflows are provided to understand and refer to while building your own agents.

Follow the instructions below to run the agents:
``` bash
python3 Agents/simplebot.py
python3 Agents/memory_agent.py
python3 Agents/toolagent.py
```
Follow the instructions in the code to run the graphs:
```bash
python3 Graphs/conditionalagent.py
python3 Graphs/loopingagent.py
python3 Graphs/multinodeagent.py
python3 Graphs/multipleinputagent.py
python3 Graphs/helloworldagent.py
```
